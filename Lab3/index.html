RobotArm - приложение для управления роботизированными объектами.

Имеется роботизированная рука которая может показывать жесты русского жестового языка, есть необходимость управлять ей дистанционно с мобильного устройства.
Скачать приложение

О проекте

Подключение к руке будет осуществлятся через bluetooth. Также будет реализована поддержка устройств с bluetooth low energy. В приложении будет возможность выводить как жесты по категориям, так и демонстрации жестов букв. Будет использоваться система двухэтпаной аутентификаций: Сначала пин-код для доступа к приложению, затем пароль для подкдючения к руке. В разработке используется самые передовые стредства и сервисы: Flutter SDK, Figma, Android Studio, Lunacy, WebStorm, Trello, GitHub.
